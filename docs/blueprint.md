# **App Name**: WishStream

## Core Features:

- Add & Manage Wishlist Items: Users can add new items to their wishlist including a name, an optional description, and an external URL. Existing items can be viewed, edited, or removed.
- Persistent Wishlist Storage: User's wishlist items are automatically saved and loaded to ensure continuity across sessions.
- Dynamic User Profile Display: The left sidebar will display the user's avatar and birthdate, with interactive micro-animations.
- AI-Powered Item Enhancement Tool: Utilize an AI tool to suggest concise descriptions or relevant tags for wishlist items based on their title or provided URL.
- Clean List Layout: Wishlist items are presented in a clean, scrollable list format on the right panel of the application.

## Style Guidelines:

- Primary element color: A deep, cool grey (#787D88) for headings and interactive elements, conveying professionalism and stability.
- Background color: An extremely light, almost white, cool grey (#F6F7F9), ensuring a clean and expansive canvas.
- Accent color: A soft yet distinct blue (#439DD8), providing subtle highlights and interaction cues while maintaining a monochrome feel.
- Body and headline font: 'Inter' (sans-serif) for its modern, objective, and clean aesthetic, suitable for all text roles.
- Minimal, clean line icons that complement the overall modern and uncluttered design.
- Two-column split layout with a fixed left sidebar dedicated to dynamic user profile details, and a scrollable right panel containing the wishlist items.
- Subtle micro-animations around user profile elements (avatar, birthdate) and smooth transitions for adding, editing, or removing wishlist items.
